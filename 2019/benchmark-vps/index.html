Benchmark VPS

Written by MFR with ♥ on 20 October 2019 in linux server 1 min

Salah satu cara untuk benchmark vps dengan menggunakan script nench berikut.

$ (curl -s bench.wget.racing | bash; curl -s bench.wget.racing | bash) 2>&1 | tee nench.log
Output-nya berupa file nench.log
