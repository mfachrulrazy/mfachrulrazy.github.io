Mengatur Waktu pada Postgresql

Written by MFR with ♥ on 7 January 2019 in database postgresql 1 min

Pengaturan waktu pada postgresql dapat dilakukan dengan cara berikut.
Melihat daftar time zone

select * from pg_timezone_names;
Atur waktu sesuai dengan zona
set timezone = 'Asia/Makassar';
Cek perubahan
select now();
Jika ingin perubahan secara permanen bisa dengan mengubah file postgresql.conf
...
log_timezone = 'Asia/Makassar'
...
timezone = 'Asia/Makassar'
Kemudian restart servis postgresql
