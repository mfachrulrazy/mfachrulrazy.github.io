Upgrade Hugo di Windows

Written by MFR with ♥ on 6 April 2019 in hugo blog windows 1 min

Upgrade Hugo dapat dilakukan dengan paket manager choco yang telah diinstall pada windows.

> choco upgrade hugo
Kemudian Ketik Y untuk menjalankan script upgrade hugo.

Install Hugo-extended

Hugo extended dapat bekerja dengan native Sass/Scss. Instalasi hugo extended pada windows dapat menggunakan choco

> choco install hugo-extended
Kemudian Ketik Y untuk menjalankan script install hugo-extended.
