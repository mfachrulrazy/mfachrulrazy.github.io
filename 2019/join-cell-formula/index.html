Formula Concat dan Vlookup pada Excel

Written by MFR with ♥ on 20 February 2019 in windows excel 1 min

Rumus untuk menggabungkan isi cell dalam excel dengan menggunakan vlookup dan concat.
Di sini ada 2 sheet yaitu:
1. Sheet1 untuk menyimpan hasil dari vlookup dan concat dari sheet berikutnya.


2. Daftar Barang adalah tabel referensi untuk sheet1.

Rumus Sheet1

Rumus untuk mengambil nama server 

=IFERROR(VLOOKUP(C3;'Daftar Barang'!$A$4:$B$50;2;FALSE);"")
Rumus untuk menggabungkan isi cell dari vlookup 
=IFERROR(CONCATENATE(VLOOKUP(D3;'Daftar Barang'!$D$4:$E$28;2;FALSE);" >> ";VLOOKUP(E3;'Daftar Barang'!$D$4:$E$28;2;FALSE));"")
