Menjalankan VM Vbox melalui File

Written by MFR with ♥ on 28 May 2019 in mesin virtual 1 min

Menjalankan mesin virtual dari sebuah file bat agar dapat lebih mudah dibanding harus membuka aplikasi virtual box lalu menjalankan salah satu mesin virtual.
Pastikan VirtualBox sudah ada di dalam environment variable Windows. Melihat daftar VM

VBoxManage list vms
Melihat daftar VM yang sedang berjalan
VBoxManage list runningvms
Untuk menjalankan VM, buat file bat dan masukkan script berikut
@echo off
VBoxManage startvm nama_vm
Jika ingin menjalankan vm tanpa tampilan jendela vm bisa menambahkan parameter --type headless
@echo off
VBoxManage startvm nama_vm --type headless
Baris pertama @echo off agar perintahnya tidak menampilkan perintah pada script. Berikut hasil perintah ketika dijalankan
Waiting for VM "nama_vm" to power on...
VM "nama_vm" has been successfully started.
