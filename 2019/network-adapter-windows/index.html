Disable/Enable Network Adapter pada Windows

Written by MFR with ♥ on 2 November 2019 in windows 1 min

Terkadang ketika kita memiliki banyak adapter, akan sangat merepotkan untuk disable/enable masing-masing adapter. Contohnya seperti ini.

Untuk memudahkan enable dan disable network adapter tersebut, kita bisa menggunakan script sederhana di PowerShell.
1. Pertama pastikan powershell dijalankan dengan mode administrator.
2. Kemudian untuk cek daftar network adapter
> Get-NetAdapter
3. Filtering bisa menggunakan Name atau InterfaceDescription.
4. Untuk enable dan disable bisa menggunakan perintah berikut. Misalkan untuk interface VirtualBox
> Enable-NetAdapter -InterfaceDescription VirtualBox*
...
> Disable-NetAdapter -InterfaceDescription VirtualBox*
