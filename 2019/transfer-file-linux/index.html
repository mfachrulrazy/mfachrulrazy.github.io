Transfer File pada Ubuntu

Written by MFR with ♥ on 25 April 2019 in linux ubuntu 1 min

Transfer file antar mesin linux bisa menggunakan tools scp.
Jika ingin mengirim file dari serverB dengan port ssh 22123 ke serverA dan login dari serverA.

# scp -P 22123 -v user@serverB:/folder/file /folder_serverA
Opsi -P untuk port dan -v untuk verbose.

Untuk menyalin file dari ServerB (login di ServerB) ke ServerA

# scp -rpv /folder_serverB user@serverA:/folder/file

Opsi -r recursive, -p preserve mode

Menjalankan Secara background

Jika kita menggunakan scp dan ingin menutup session ssh, maka scp juga akan terputus. Oleh karena itu scp akan kita jalankan di background proses dengan menggunakan nohup.
Ketika scp dijalankan biasanya akan meminta password (jika belum bertukar ssh key) oleh karena itu gunakan perintah di bawah jika ingin menampilkan permintaan password.

# nohup scp -P 22123 -v user@serverB:/folder/file /folder_serverA > nohup.out 2>&1
Setelah itu akan muncul permintaan password, kemudian tekan Ctrl+Z lalu masukkan perintah
# bg
Hal ini akan menjalankan perintah nohup secara background
nohup scp -P 22123 -v user@serverB:/folder/file /folder_serverA > nohup.out 2>&1
Untuk melihat proses download bisa monitoring dengan netstat
# netstat -tupn
