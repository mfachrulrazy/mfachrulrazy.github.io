Restore Database SQL Server

Written by MFR with ♥ on 3 April 2019 in database sql server 1 min

Proses restore database berhenti dan tidak berubah status.

Solusi

Gunakan t-sql dan masukkan kueri untuk restore database dengan menggunakan opsi with recovery 

RESTORE DATABASE NamaDB FROM DISK = 'Path\to\file.bak'
WITH RECOVERY
GO
