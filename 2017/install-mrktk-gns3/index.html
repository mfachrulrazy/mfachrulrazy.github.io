Install Mikrotik di GNS 3

Written by MFR with ♥ on 17 March 2017 in networking 2 min

Terdapat beberapa cara untuk membuat simulasi salah satunya dengan menggunakan qemu.
1. Pastikan file iso mikrotik telah di-download
2. Pertama buat image dari file iso mikrotik dengan qemu dan simpan di drive D:

C:\Program Files\GNS3\qemu-2.4.0>qemu-img.exe create -f qcow2 D:/mikrotik.img 256M

3. Setelah itu, install file img yang sudah dibuat menggunakan iso file mikrotik (file iso sudah di-rename mikrotik.iso)
4. Sesuaikan bit aplikasi qemu dengan iso yang akan dijalankan
- qemu-system-i386w.exe : 32bit
- qemu-system-x86_64w.exe : 64bit
C:\Program Files\GNS3\qemu-2.4.0>qemu-system-i386w.exe "D:/mikrotik.img" -boot d -cdrom "D:/mikrotik.iso”
5. Setelah menjalankan perintah di atas, maka proses instalasi mikrotik berjalan dan install mikrotik seperti biasa. 6. Kemudian untuk mencoba image yang sudah dibuat ketikkan perintah

C:\Program Files\GNS3\qemu-2.4.0>qemu-system-i386w.exe "D:/mikrotik.img" -boot c

7. Setelah itu, masuk buat VM-nya di GNS3 dengan masuk ke menu Edit > Preferences 8. Pilih Qemu VMs > klik New 9. Buat nama VM-nya 10. Pilih qemu binary dan total RAM yang akan digunakan 11. Pilih tipe consolenya (telnet) 12. Pilih tipe image-nya 13. Kemudian klik finish

Menambahkan interface pada Mikrotik

Pilih template mikrotik > configure template

Pada tab network > masukkan jumlah interface yang diinginkan pada Adapters Untuk mengganti icon dari mikrotik, pilih VM yang akan diganti kemudian pada tab General Settings > Pilih Symbol: > pilih symbol yang diinginkan

Troubleshoot Mikrotik tidak dapat ping ke jaringan lain

Mikrotik terdeteksi di winbox, tetapi tidak dapat terkoneksi ke jaringan lain.
Berikut topologinya

Jika mikrotik tidak dapat terkoneksi satu sama lain, salah satu penyebabnya adalah file image yang tidak bagus. Setelah dicoba untuk menggunakan file image lain, mikrotik dapat melakukan tes ping satu sama lain.
