Mounting eksternal Drive di Linux

Written by MFR with ♥ on 27 March 2017 in linux centos 1 min

Mounting dengan file system ntfs.

Pertama melihat daftar disk

# fdisk -l

Cek filesystem
# file -s /dev/sdxx

Buat direktori mount point
# mkdir /mnt/externalhdd

Mounting
# mount -t ntfs-3g /dev/sdxx /mnt/externalhdd

Unmounting (eject):

# umount /dev/sdxx
