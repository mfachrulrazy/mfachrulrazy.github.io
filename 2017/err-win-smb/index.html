Windows 10 Tidak dapat akses SMB

Written by MFR with ♥ on 14 November 2017 in windows 1 min

Windows 10 Enterprise secara default melarang fitur SMB. Penjelasan lebih lanjut langsung dari situs micorosoft

Guest access in SMB2 disabled by default in Windows 10 Fall Creators Update and Windows Server 2016 version 1709

Caranya ubah melalui Group Policy Settings

Computer configuration\administrative templates\network\Lanman Workstation

Aktifkan Enable insecure guest logons

Dengan catatan, jika mengaktifkan mode ini, akan mengurangi keamanan pada Windows klien. Mungkin hal ini terjadi akibat virus wannacry yang terjadi.
