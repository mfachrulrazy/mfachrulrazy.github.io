Error Monitoring PRTG HTTPS

Written by MFR with ♥ on 12 July 2017 in monitoring networking 1 min

Ketika mengatur PRTG menjadi https, web servernya tidak berhasil berjalan. Pesan error yang muncul ketika dibuka kurang lebih sebagai berikut:

Connection Refused Socket Error # 10061

Penyebabnya adalah pada saat penggantian koneksi menjadi https, maka portnya juga berganti. Oleh karena itu harus diganti juga pada bagian PRTG Console Setting dan PRTG Administration Tool.

  1. Buka PRTG dan masuk ke PRTG Administration Tool
  2. Berikut contoh konfigurasinya
  3. Pengaturan perubahan konfigurasi dari http ke https
