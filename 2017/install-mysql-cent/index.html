Install Mysql Centos 7

Written by MFR with ♥ on 2 November 2017 in mysql database centos 1 min

Instalasi mysql versi community pada centos dapat dilakukan dengan cara download paket mysql terlebih dahulu dari repository mysql.

  1. Download dan tambahkan repo. 
    # wget http://repo.mysql.com/mysql57-community-release-el7-11.noarch.rpm
# sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm
# yum update
  2. Install mysql dan jalankan servisnya 
    # yum install mysql-server
# systemctl start mysqld
  3. Jalankan script mysql_secure_installation setelah selesai menginstall mysql 
    # mysql_secure_installation

Install Mysql Client

  1. Download file repositori mysql dari situs resminya (versi mysql community) 
    # wget https://repo.mysql.com//mysql57-community-release-el7-11.noarch.rpm
  2. Install paket tersebut ke dalam RPM paket centos 
    # yum localinstall mysql57-community-release-el7-11.noarch.rpm
  3. Cek repository telah terinstall 
    # yum repolist enabled | grep "mysql.*-community.*"
  4. Kemudian Install mysql client 
    # yum install mysql-community-client

Uninstall Mysql Server

  1. Lihat daftar paket mysql 
    # yum list installed | grep mysql
  2. Kemudian uninstall seluruh paket mysql yang telah terinstall 
    # yum remove mysql-community-client mysql-community-server mysql-community-common mysql-community-libs mysql-community-libs-compat
  3. Dapat juga menghapus dengan paket --remove-leaves 
    # yum install yum-plugin-remove-with-leaves
# yum remove mysql-server --remove-leaves
  4. Hapus direktori mysql 
    # rm -rf /var/lib/mysql/
# rm -rf /etc/my.cnf
  5. Cek apakah mysql masih ada atau belum 
    # find / “mysql”

File-file Mysql

  1. File konfigurasi berada di /etc/my.cnf
  2. File histori mysql dari perintah yang digunakan untuk mysql ~/.mysql-history (folder file history disimpan tergantung dari user mana ketika kita input perintah-perintah di mysql)
