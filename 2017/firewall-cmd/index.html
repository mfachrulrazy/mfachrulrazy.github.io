Firewall CMD Centos

Firewall-CMD

Firewalld adalah sebuah tools untuk mengatur firewall pada Linux. Berikut beberapa perintah firewall-cmd yang biasa digunakan.
Melihat status firewall

# firewall-cmd --state

Preconfigured zone dapat dilihat pada /usr/lib/firewalld/zones
Melihat zone yang ada
# firewall-cmd --get-zones
Setiap network adapter interface hanya dapat didaftarkan pada satu zone Melihat rule firewall
# firewall-cmd --list-all-zones
# firewall-cmd --list-all-services
# firewall-cmd --list-all
# firewall-cmd --list-ports

Menambahkan servis berdasarkan zona publik
# firewall-cmd --zone=public --add-service=http

Menambahkan servis sercara permanent
# firewall-cmd --permanent --add-service=http
# firewall-cmd --permanent --add-service=https

Menambahkan port secara permanent
# firewall-cmd --permanent --add-port=2323/tcp

Menyimpan rule firewall
# firewall-cmd --reload

Menghapus rule firewall
# firewall-cmd --zone=public --remove-port=2323/tcp
# firewall-cmd --runtime-to-permanent 
# firewall-cmd --reload
