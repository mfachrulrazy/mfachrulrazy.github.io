Install Nodejs Centos 7

Written by MFR with ♥ on 30 September 2017 in linux server centos 1 min

Nodejs adalah salah satu framework javascript yang berjalan pada sisi server (server side). Kali ini nodejs akan diinstall pada Centos 7.
1. Download node js melalui situs resminya 

# wget http://nodejs.org/dist/v0.10.30/node-v0.10.30.tar.gz
2. Ekstrak file yang telah didownload dan masuk ke direktori hasil ekstrak 
# tar xzvf node-v* && cd node-v*
3. Install gcc dan c++ untuk melakukan compile 
# yum install gcc gcc-c++
4. Konfigurasi dan lakukan compile 
# ./configure
# make
# make install
5. Setelah selesai, cek versi nodejs 
# node --version
6. Jika server centos memakai firewall, maka kita harus membuka port 7000 (port nodejs) pada firewalld 
# firewall-cmd --permanent --add-port=7000/tcp
# firewall-cmd --reload
7. Cek port-nya jika telah ditambahkan 
# firewall-cmd –list-port
