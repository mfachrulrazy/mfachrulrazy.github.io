SSH key Akun Github

Written by MFR with ♥ on 21 July 2017 in devops git 1 min

Menambahkan SSH Key ke Akun github dari PC tujuannya adalah kita tidak perlu lagi memasukkan password tiap kali ingin terhubung dengan akun github. Berikut caranya

  1. Pastikan git sudah terinstall pada PC
  2. Buat ssh key di cmd dengan
    > ssh-keygen
  3. Ssh key akan secara default masuk ke folder ~user/.ssh/*
  4. Ada 2 file yang dibuat seperti di linux yaitu id_rsa (private key) dan id_rsa.pub (public key)
  5. Copy isi file id_rsa.pub
  6. Masuk ke menu Settings > SSH and GPG Keys pada akun github
  7. Klik New SSH Key > tambahkan title dan paste isi dari id_rsa.pub yang telah di-copy ke dalam Key > Klik Add SSH Key
  8. Setelah itu konfirmasi password kembali dan ssh telah berhasil ditambahkan
