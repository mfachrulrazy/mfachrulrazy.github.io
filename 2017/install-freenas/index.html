Install dan Konfigurasi Free NAS

Written by MFR with ♥ on 20 July 2017 in server linux 1 min

Free NAS merupakan salah satu sistem operasi yang khusus dibuat untuk keperluan Network Attached Storage. Free NAS merupakan turunan dari freeBSD yang mendukung CIFS, NFS, iSCSI, FTP, RSYN bersama dengan RAID.
1. Untuk proses instalasi sendiri seperti instalasi linux pada umumnya, yang membutuhkan partisi root ( / ) dan swap.
2. Pada saat masuk ke konfigurasi awal, bisa keluar terlebih dahulu.
3. Untuk menambah user dan grup bisa melalui menu Account > Groups > Add Group dan Account > Users > Add User
4. Menambahkan storage dari menu Storage > Volumes > Volume Manager

5. Buat dataset di Storage > Volumes > /mnt/vol01 > Create Dataset6. Konfigurasi dataset7. Kemudian konfigurasi permissions8. Buat kembali dataset dengan nama awalan NFS_ (untuk sistem UNIX).
9. Kemudian atur permission seperti gambar berikut10. Buat sharing NFS11. Masukkan path direktorinya12. Kemudian tambahkan juga untuk sharing OS Windows > masuk di Add Windows SMB Share
13. Masukkan path seperti pada Unix, tetapi pilih yang CIFS
14. Setelah selesai, jalankan pada windows explorer (satu network) dengan cara akses SMB
