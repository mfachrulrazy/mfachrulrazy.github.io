Apache dan SE Linux

Written by MFR with ♥ on 29 August 2017 in web server apache security 1 min

Secara default keamanan di centos mencegah script Apache untuk mengakses database. Untuk mengaktifkannya lakukan:

# setsebool httpd_can_network_connect_db=1
Jika ingin persistent pada saat server di-restart tambahkan opsi -P 
# setsebool -P httpd_can_network_connect_db on
Cek hasil konfigurasi 
# getsebool httpd_can_network_connect_db
