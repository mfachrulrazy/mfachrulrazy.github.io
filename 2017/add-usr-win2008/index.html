Standard User Shutdown Windows Server 2008

Written by MFR with ♥ on 21 September 2017 in server windows 1 min

Menambahkan user biasa untuk shutdown pada Windows Server 2008.
1. Masuk ke Menu Administrative Tools > Local Security Policy.
2. Jika melalui Windows Run bisa dengan perintah berikut.

secpol.msc
3. Pada Jendela Local Security Policy pilih menu.
Local policies > User Rights Assignment > Shut down the system
4. Pilih menu tersebut > pilih Add User or Group > Tambahkan User.
5. Klik Ok dan Selesai.
