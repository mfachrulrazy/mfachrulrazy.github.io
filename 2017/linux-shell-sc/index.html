Linux Shell Shortcut

Written by MFR with ♥ on 13 June 2017 in linux centos 1 min

Berikut beberapa shortcut yang sering digunakan dalam shell linux.
Berpindah ke akhir baris 

Ctrl + E
Berpindah ke awal baris 
Ctrl + A
Maju satu kata 
Esc + F
Mundur satu kata 
Esc + B
Keluar secara paksa dari aplikasi 
Ctrl + C
Suspend aplikasi 
Ctrl + Z
Menjalankan aplikasi di foreground 
# fg
Menjalankan aplikasi di background 
# bg
Untuk lebih lengkapnya bisa cek manual dari bash 
# man bash
