Pengaturan Waktu Centos 7

Written by MFR with ♥ on 27 March 2017 in centos linux 1 min

Pengaturan waktu dan tanggal pada Centos bisa dengan menggunakan tools timedatectl.
Untuk melihat pengaturan waktu yang terpasang 

# timedatectl
Mengatur tanggal
# timedatectl set-time [YYYY-MM-DD]
Mengatur waktu gunakan format 24H
# timedatectl set-time [HHSS]
Pengaturan waktu berdasarkan waktu lokal
# timedatectl set-local-rtc yes
Pengaturan waktu berdasarkan UTC
# timedatectl set-local-rtc no
Melihat zona lokasi
# timedatectl list-timezones
...
Asia/Makassar
...
Mengatur zona lokasi
# timedatectl set-timezone Asia/Makassar
Mengatur waktu dengan menggunakan NTP 
# timedatectl set-ntp yes
Tanpa menggunakan NTP 
# timedatectl set-ntp no
