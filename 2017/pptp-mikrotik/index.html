VPN PPTP pada Mikrotik

Written by MFR with ♥ on 1 May 2017 in networking mikrotik 1 min

Kadangkala ketika kita berada di luar kantor (tidak terkoneksi dengan jaringan lokal kantor), kita ingin me-remote perangkat tertentu yang berada di kantor. Salah satu cara agar bisa membuat koneksi seolah-olah berada di jaringan kantor adalah VPN. Pada postingan kali ini VPN yang digunakan adalah PPTP dari Mikrotik. Berikut langkah instalasinya.
1. Pastikan paket PPP telah terinstall di Mikrotik System > Packages

2. Kemudian masuk pada menu PPP > tab Interfaces > klik PPTP Server > centang pada bagian Enabled dan pastikan pada bagian Default Profile menggunakan default-encryption3. Kemudian tambahkan user yang akan mengakses PPTP server tersebut. Pada tab Secrets > klik tanda +

Name: username
Password: password
Services: untuk services pilih yang ‘any’ saja.
Profile: pilih default-encryption
Local Address: IP yang akan digunakan PPTP Server (satu jaringan dengan remote address)
Remote Address: IP yang akan digunakan oleh PPTP Client

4. Setelah itu, lakukan konfigurasi PPTP client pada PC yang akan mengakses mikrotik tersebut.
5. Untuk pengaturan PPTP pada Windows 10, pada bagian server name, isikan dengan IP publik Mikrotik.6. Contoh PPTP Interface telah aktif7. Jika terdapat firewall, allow paket tcp dari PPTP dan port-nya pada interface yang ke internet.
