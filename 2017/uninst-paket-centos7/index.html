Uninstall Paket Linux

Written by MFR with ♥ on 20 November 2017 in server centos ubuntu linux 2 min

Untuk menghapus seluruh paket bisa menggunakan perintah yum remove pada centos atau apt-get remove pada ubuntu. Berikut contoh untuk menghapus paket secara keseluruhan (completely remove package)

Menghapus paket Apache/Httpd

  1. Lihat paket yang terinstall 
    # yum list installed *httpd*
  2. Lihat paket mod_ yang terinstall dan lihat paket php 
    # yum list installed *mod_*
# yum list installed php*
  3. Jika paket tidak didapat
  4. Jika ada paket yang terinstall
  5. Tanda bintang di awal dan di akhir dapat disesuaikan.
  6. Untuk melihat lokasi httpd secara keseluruhan 
    # rpm –ql httpd
  7. Melihat info httpd 
    # yum info httpd
  8. Remove httpd sesuai dengan paket yang terinstall 
    # yum remove httpd httpd-tools
  9. Hapus user apache 
    # userdel –r apache
  10. Hapus directory dari apache 
    # rm –rfv /var/www
# rm –rfv /etc/httpd
# rm –rfv /usr/lib/httpd

Menghapus paket PHP

  1. Menghapus paket php secara menyeluruh 
    # yum -y remove php*
  2. Jika ingin menghapus seluruh paket php

Menghapus paket Mariadb di Ubuntu

  1. Pertama hentikan service mysql 
    $ sudo service mysql stop
  2. Kemudian uninstall menggunakan perintah berikut 
    $ sudo apt-get –purge remove “mysql*”
  3. Hapus file konfigurasi 
    $ sudo rm -rf /etc/mysql
  4. Lakukan autoremove dan autoclean untuk menghapus sisa paket dari mariadb 
    $ sudo apt-get autoremove
$ sudo apt-get autoclean
  5. Untuk mengecek sisa file yang terkait mariadb 
    $ locate mysql

Menghapus paket Java di Ubuntu

  1. Untuk menghapus java lakukan cek versi java terlebih dahulu apakah jre atau jdk dengan perintah berikut 
    # rpm -qa | grep java
Atau
# rpm -qa | grep jdk
Atau
# java --version
Atau
# update-alternatives --config java
  2. Untuk mengahpus gunakan perintah 
    # rpm -e <nama paket java yang terinstall>
  3. Lakukan pengujian
