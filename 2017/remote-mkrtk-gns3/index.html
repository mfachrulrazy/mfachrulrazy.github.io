Remote Mikrotik Gns3 via Winbox

Written by MFR with ♥ on 19 November 2017 in networking remote 1 min

Untuk terhubung dengan mikrotik di GNS3 dapat menggunakan interface loopback.
Gunakan cloud untuk menghubungkan interface jaringan komputer ke winbox gns 3 > klik kanan Configure

Kemudian perhatikan bagian Ethernet Interfaces terdapat Npcap Loopback Adapter
Perhatikan bagian interface di sistem operasi (windows)
