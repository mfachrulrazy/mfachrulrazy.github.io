Servis Wireless tidak berjalan pada Windows 7

Written by MFR with ♥ on 12 March 2017 in windows wireless networking 1 min

Kemarin dapat masalah dari salah satu client yaitu servis wireless-nya tidak berjalan di windows 7 32bit.
Setelah dilakukan troubleshooting

Ada beberapa solusi untuk mengatasi hal ini: * Repair windows 7 * Install ulang * Cek services WLAN Auto Config berjalan apa tidakJika tidak berjalan, pastikan services yang ada di dependencies berjalan.
Sebelumnya pastikan terlebih dahulu pada regedit bagian berikut:

HKEY_LOCAL_MACHINE/System/ControlSet001/Services/Ndisuio 
HKEY_LOCAL_MACHINE/System/ControlSet002/Services/Ndisuio 
HKEY_LOCAL_MACHINE/CurrentControlSet/Services/Ndisuio

Pastikan, keys untuk Start bernilai 3
Kemudian setelah itu, pastikan juga file xml yang di-load oleh windows di folder
C:\Windows\L2Schemas ukurannya tidak 0 kb (zeroed bytes)

Bisa menggunakan tools monitoring untuk melihat services apa yang berjalan pada suatu program
