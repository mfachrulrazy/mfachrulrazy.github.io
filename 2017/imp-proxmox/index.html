All about Proxmox

Written by MFR with ♥ on 12 July 2017 in linux proxmox networking 1 min

Postingan ini akan membahas mengenai beberapa hal yang telah saya lakukan dengan Proxmox.

Menghapus pesan No Valid Subscription

Sejak versi proxmox 4.x ke atas biasanya pada saat setelah login di proxmox, langsung muncul No Valid Subscription. Untuk menghapusnya bisa dengan cara berikut

  1. Edit file /usr/share/pve-manager/ext4/pvemanagerlib.js
  2. Ingat untuk membuat backup file tersebut terlebih dahulu
    • proxmox 4.4 foldernya di: /usr/share/pve-manager/ext6
    • proxmox 5 foldernya di: /usr/share/pve-manager/js
  3. Kemudian edit pada baris berikut 
    if (data.status !== 'Active') { ...
// ubah menjadi
if (false) { ...
  4. Simpan perubahannya, dan pesan tersebut tidak akan muncul kembali.

Mengganti Hostname Proxmox

Untuk mengganti hostname proxmox perlu diperhatikan bahwa mesin vm melekat pada nama server sebelumnya. Jika mengganti hostname secara langsung, maka akibatnya mesin VM akan tetap berada pada hostname sebelumnya.

Hal ini terjadi karena hostname hanya diganti file hostname-nya. Oleh karena itu, Untuk mengganti hostname perlu dilakukan langkah berikut:

  1. Edit file hostname dan hosts sesuai dengan nama hostname 
    # vi /etc/hostname
# vi /etc/hosts
  2. Kemudian restart service pve-nya 
    # service pve-cluster restart
  3. Setelah itu akan terdapat folder baru sesuai dengan nama hostname di: /etc/pve/nodes/[newhostname]
  4. Kemudian copy file .conf yang ada dalam folder host lama.
    • Untuk VM di folder: qemu-server
    • Untuk CT di folder: openvz

Folder iso proxmox

Identifikasi letak folder penyimpanan iso dari proxmox ada di /var/lib/vz/template/iso
