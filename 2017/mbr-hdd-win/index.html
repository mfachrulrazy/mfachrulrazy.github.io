Konversi Tipe Harddisk pada Windows

Written by MFR with ♥ on 3 July 2017 in windows 1 min

Sebelum melakukan konversi HD, backup seluruh data yang ada di dalamnya, karena konversi mbr ke gpt akan melakukan pembersihan isi seluruh harddisk. Setelah data di-backup lakukan langkah berikut:
1. Matikan pc, dan colok usb installer windows
2. Nyalakan pc dan booting ke usb 3. Setelah itu pada saat di jendela Windows Setup tekan Shift + F10 untuk masuk ke command prompt

> diskpart
4. Identifikasi drive untuk reformat
> list disk
5. Pilih drive, dan reformat
> select disk <disk number>
> clean
> convert gpt
> exit
6. Tutup cmd dan lanjutkan instalasi Windows.
7. Pada saat pemilihan tipe instalasi, pilih Custom. Drive akan muncul sebagai unallocated space
