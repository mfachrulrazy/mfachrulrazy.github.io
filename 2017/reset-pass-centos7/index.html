Reset Password Centos 7

Written by MFR with ♥ on 10 May 2017 in linux open source centos 1 min
  1. Pada saat booting pertama kali dan muncul GRUB menu, tekan e
  2. Kemudian ubah pada bagian ro menjadi rw dan tambahkan init=/sysroot/bin/sh
  3. Setelah itu tekan ctrl + x untuk masuk ke single user mode.
  4. Tunggu sebentar untuk booting, kemudian akses sitem dengan ketikkan perintah berikut:
    :/# chroot /sysroot
  5. kemudian reset password
    :/# passwd root
  6. Peringatan “the password fails the dictionary check” disebabkan karena password terlalu lemah, namun password tetap bisa diterapkan.
  7. Konfirmasi password root baru agar parameter SELinux ikut berubah dengan perintah
    :/# touch /.autorelabel
  8. Kemudian keluar dari chroot dan reboot mesin
    :/# exit  
:/# reboot

Reset password Root via Terminal

Mengganti password root dari terminal tinggal menggunakan perintah berikut

sudo -i passwd

Kemudian masukkan password baru untuk user root.
