Hugo adalah salah satu static framework untuk membuat website atau biasa dikenal dengan istilah framework static site generators. Untuk menginstall hugo di Windows, choco paket manajer harus terlebih dahulu terinstall. Install Linux dari Windows Store

Karena Windows mulai menyukai Linux

 Posted on April 2, 2018  |  1 minutes  |  MFR

  1. Pertama harus mengaktifkan WSL atau Windows Subsystem for Linux
  2. Caranya, buka powershell sebagai Administrator dan jalankan perintah berikut:
    > Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
  3. Restart PC
  4. Kemudian install Linux dari Windows Store
