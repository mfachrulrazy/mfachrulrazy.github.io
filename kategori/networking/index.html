Konfigurasi Nmtui Centos 7

Konfigurasi jaringan dengan Nmtui pada centos 7

 Posted on February 17, 2018  |  1 minutes  |  MFR

Waktu itu saya mencoba menginstall centos di virtual machine dan lupa menambahkan adapter jaringan. Jadi mesin virtual dimatikan terlebih dahulu kemudian tambah adapter jaringan pada virtual box.
Tapi masalahnya yaitu file network di config tidak muncul. Hal tersebut dapat ditangani dengan nmtui.

  1. Jalankan nmtui dari terminal 
    # nmtui
  2. Pilih edit connection > kemudian pilih Wired connection 1 > pilih Edit
  3. Kemudian ubah nama profil dan IPv4 configuration menjadi manual
  4. Kemudian OK > Back > Quit
  5. Restart service network

Maximum Transmission Unit

 Posted on April 28, 2017  |  1 minutes  |  MFR

Kemarin penasaran dengan apa itu MTU pada saat menganalisis jaringan di kantor. Jadi, Maximum Transmission Unit adalah istilah dalam teknologi informasi yang merujuk kepada ukuran paket data terbesar yang dapat ditransmisikan melalui sebuah media jaringan. Ukuran MTU adalah bervariasi, tergantung teknologi jaringan yang digunakan. Contohnya adalah dalam jaringan berbasis teknologi Ethernet, ukuran MTU maksimum adalah 1500 bytes.
