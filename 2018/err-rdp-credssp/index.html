Error Remote Desktop Connection

Written by MFR with ♥ on 19 May 2018 in remote windows 1 min

Terjadi error Pada saat mencoba remote desktop ke salah satu server windows di kantor.

Versi windows yang digunakan

Penyebab

Setelah membaca beberapa artikel, ini adalah kelemahan dari RDP. Pada bulan Mei 2018 microsoft telah mengeluarkan update security untuk RDP ini. Jadi ketika ingin melakukan remote desktop, komputer sumber dan tujuan harus ter-update untuk security di RDP ini.

Penyelesaian

  1. Pertama Cek update dari pc client terlebih dahulu (yang me-remote)
  2. Jika update sudah terinstall di client tetapi muncul error, berarti update belum terinstall di server.
  3. Penyelesaian sementara, bisa menggunakan ketikkan perintah berikut dari command prompt (administrator) 
    > REG  ADD HKLM\Software\Microsoft\Windows\CurrentVersion\Policies\System\CredSSP\Parameters\ /v AllowEncryptionOracle /t REG_DWORD /d 2
