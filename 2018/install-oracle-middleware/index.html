Install Oracle Fusion Middleware

Written by MFR with ♥ on 20 March 2018 in oracle server 1 min

Instalasi Oracle Fusion Middleware untuk kebutuhan testing kantor. Persiapan yang diperlukan:

  • Linux OS with GUI (dalam hal ini menggunakan Centos 7 GUI)
  • JDK versi 1.8xxx
  • File installer oracle fmw_12.2.1.3.0_wls_quick.jar

Setelah semua kebutuhan siap, lanjut ke proses instalasi.

  1. Jalankan file master oracle

    # java -jar fmw_12.2.1.3.0_wls_quick.jar

  2. Pemilihan inventory setup

  3. Atur direktori di /home/laki/oraInventory

  4. Cek prerequisite-nya

  5. Pada jendela welcome klik next

  6. Kemudian skip auto updates klik next

  7. Lokasi direktori oracle yang akan diinstall

  8. Kemudian pengecekan persyaratan

  9. Kemudian klik install

  10. Klik Next kemudian Finish

  11. Setelah itu window Jdeveloper muncul dan pemilihan role

  12. Konfirmasi untuk import pengaturan sebelumnya

  13. Masukkan password untuk weblogic

  14. Kemudian Run Instance

  15. Jalankan oracle web logic dengan perintah sudo pada folder: home/user/.jdeveloper/system-xxx/DefaultDomain/bin

  16. Oracle akan menggunakan port 7101 dan 7102

Install Sertifikat

  1. Salin file sertifikat ke direktori oracle
    $ cp \*.fin.us6.oraclecloud.com.cer ~/Oracle_Home/wlserver/server/lib/
  2. Pindah ke direktori tempat file disalin
    $ cd ~/Oracle_Home/wlserver/server/lib/
  3. install sertifikat
    $ keytool -import -file \*.fin.us6.oraclecloud.com.cer -keystore DemoTrust.jks -storepass DemoTrustKeyStorePassPhrase
