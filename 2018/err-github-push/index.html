Error Github Push

Written by MFR with ♥ on 4 September 2018 in devops git 1 min

Pada saat ingin melakukan push ke repo github terdapat error. 

...
error: src refspec master does not match any.
error: failed to push some refs to 'git@github.com:user/repo.git'

Solusi

Masukkan identitas terlebih dahulu pada git server. 

# git config --global user.email "user@email.com"
Kemudian lakukan commit ulang dan push ke repo.
