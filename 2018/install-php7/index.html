Install PHP 7.0 dan Modul SQL Server pada Centos

Written by MFR with ♥ on 27 September 2018 in php server centos sql server 1 min
  1. Install paket-paket yang dibutuhkan oleh php dan sql server menggunakan repo remi
    # yum --enablerepo=remi-safe -y install php70 php70-php-pear php70-php-mbstring php70-php-pdo php70-php-devel php70-php-mysql
  2. Install paket development tools
    # yum -y groups install "Development Tools"
  3. Enable php 70
    # scl enable php70 bash
  4. Menjalankan php70 pada saat login pertama kali dengan cara membuat file script di /etc/profile.d/php70.sh
    # create new
#!/bin/bash
source /opt/remi/php70/enable
export X_SCLS="`scl enable php70 'echo $X_SCLS'`"
  5. Install paket unixodbc-devel
    # yum install unixODBC-devel
  6. Install sql server dan pdo menggunakan pecl
    # pecl install sqlsrv pdo_sqlsrv
  7. Tambahkan modul sql server dan pdo sql server di konfigurasi php
    # echo 'extension=pdo_sqlsrv.so' > /etc/opt/remi/php70/php.d/30-pdo_sqlsrv.ini
# echo 'extension=sqlsrv.so' > /etc/opt/remi/php70/php.d/30-sqlsrv.ini
