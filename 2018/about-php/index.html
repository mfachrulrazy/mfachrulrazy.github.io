Pengaturan pada PHP

Written by MFR with ♥ on 25 January 2018 in php server 2 min

Menambahkan maximum upload file di PHP

Ubah ukuran maksimum file upload di php dapat dengan cara mengedit file /etc/php.ini

    ...
    post_max_size = 100M
    ...
Kemudian reload service php

Menambahkan maximum eksekusi PHP

Menambahkan maksimum eksekusi edit file /etc/php.ini

    ...
    max_execution_time = 500
    ...

Menampilkan captcha

Biasanya developer website membuat captcha pada autentikasi halaman webnya. Salah satu cara untuk memunculkannya adalah menginstall paket tambahan php yaitu php-gd.

# yum install php-gd

Install Library PHP mbstring

Sejak menggunakan php 70, paket yang diinstall harus berbeda, karena php bawaan dari centos 7 adalah php 5.4.

# yum install php70u-mbstring

Web CodeIgniter tidak dapat diakses

Adakalanya sebuah website yang dibangun di platform yang berbeda dengan server production akan mengakibatkan error.
Sebagai contoh, developer membangun websitenya di OS Windows, sedangkan server production menggunakan OS Centos. Akibatnya, kadangkala website berjalan normal di platform developer karena salah satunya adalah windows tidak case sensitive sedangkan pada server prodcution bisa berjalan.
Begitu pula di aplikasi codeigniter yang case sensitive, oleh karenanya ketika salah satu website kantor tidak berjalan, penyebabnya adalah penamanaan modules tidak sesuai.
Biasanya modul codeigniter diletakkan di folder website/apps/backend/modules tidak diawali dengan huruf besar.

PHP Error Deprecated and Warning

Salah satu website tidak dapat berjalan dan pada saat pengecekan proses php, muncul error

  • Error Warning : Dapat membuat komentar pada register_globals di /etc/php.ini
  • Error Deprecated : Upgrade versi php
