List Servis pada Windows Server

Written by MFR with ♥ on 18 May 2018 in windows server 1 min

Melihat seluruh servis yang ada pada windows server. Buka power shell dan ketikkan perintah berikut

> Get-Service
Melihat servis dengan awalan s dan di-sort berdasarkan status (running/stopped)
> Get-Service "s*" | Sort-Object status
Untuk melihat servis yang sedang berjalan
> net start
