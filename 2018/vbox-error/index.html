VM Virtual Box Error

Written by MFR with ♥ on 5 November 2018 in mesin virtual windows virtual box 1 min
  1. Waktu itu, ingin menjalankan salah satu vm di virtualbox tetapi vm tidak dapat dijalankan dan terdapat error seperti berikut ini
  2. setelah searching di internet ternyata ada konflik dengan Hyper-V
  3. Jadi solusinya, matikan fitur hypervisorlaunchtype
    > bcdedit /set hypervisorlaunchtype off

  4. Jika ingin mengembalikan nilai hypervisorlaunchtype
    > bcdedit /set {current} hypervisorlaunchtype auto

Penyebab

  • Penyebabnya adalah karena pada mesin host (utama) terinstall Hyper-V.
  • Karena kemarin sempat install docker for windows yang membutuhkan Hyper-V.
  • Belum mencoba untuk menjalankan vm di virtualbox dengan docker secara bersamaan.
