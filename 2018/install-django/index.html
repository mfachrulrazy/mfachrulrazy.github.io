Install Django

Written by MFR with ♥ on 2 June 2018 in python web server 1 min

Setelah membahas mengenai virtual environment di python, kali ini akan dibahas mengenai instalasi django yaitu salah satu web framework yang berbasis python.

  1. Pastikan python telah terinstall, dalam hal ini menggunakan python 3.

  2. Python 3 sudah termasuk dengan pip3, dapat dicek dengan perintah

    > pip3 list

  3. Setup virtual environment (menggunakan virtualenv-wrapper)

    > pip3 install virtualenvwrapper-win

  4. Buat direktori kerja untuk django dan masuk ke dalam direktori tersebut.

  5. Buat virtual environment contoh dengan nama djangomu

    > mkvirtualenv djangomu

  6. Install django dalam virtual environment

    > pip3 install django

    Pada saat melakukan perintah ini, biasanya juga akan mengecek versi tools pip3, jika ada versi terbaru maka akan muncul informasi untuk upgrade pip.

  7. Cek versi django

    > py -3 -m django --version
Atau
> py –m django --version

  8. Setelah django berhasil terinstall, selanjutnya adalah buat proyek django

    > django-admin startproject tesweb
> cd tesweb

  9. Jalankan web django

    > python manage.py runserver

  10. Selanjutnya buka browser dan jalankan localhost pada port 8000 (default django)

    Sampai di sini, kita telah berhasil menginstall django, langkah selanjutnya adalah akan kita bahas pada postingan berikutnya mengenai pembuatan skeleton proyek.
