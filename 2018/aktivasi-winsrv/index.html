Aktivasi Windows Server 2012

Written by MFR with ♥ on 25 July 2018 in windows server 1 min

Aktivasi pada Windows Server 2012 R2 Standard Edition terdapat 2 tahap. Pertama identifikasi terlebih dahulu versi windows dengan perintah berikut.

> DISM /online /Get-CurrentEdition
Ubah versi windows lalu masukkan Edisi yang diinginkan dan Product Key
> DISM /online /Set-Edition:ServerStandard /ProductKey:XXXXX-XXXXX-XXXXX-XXXXX-XXXXX /AcceptEula
Dimana xxxxx adalah serial key dari lisensi windows yang dibeli.
Setelah itu, masukkan kembali serial key melalui System > Change Product Key > masukkan serial key tersebut.
Restart Server.
