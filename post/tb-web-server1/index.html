Troubleshoot Web Server I

Troubleshoot error web server

 Posted on September 13, 2017  |  1 minutes  |  MFR

Waktu itu salah satu website di kantor bermasalah. Webnya tidak bisa diakses, tetapi servis database dan web server-nya berjalan dengan normal. Cek di firewall dalam hal ini menggunakan iptables karena os Centos 5.11 (end-of-life).
Izinkan traffic mysql dan http dengan menambahkan rule di iptables pada file /etc/sysconfig/iptables 

-A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 80 -j / ACCEPT
-A RH-Firewall-1-INPUT -m state --state NEW -m tcp -p tcp --dport 3306 -j / ACCEPT
Kemudian restart iptables-nya 
# /etc/init.d/iptables restart
