Belajar Vi dan Vim

 Posted on August 10, 2016  |  1 minutes  |  MFR

Vi adalah salah satu editor kode yang banyak digunakan oleh para sys admin dan developer dikarenakan kestabilan dan kemampuannya dalam mengedit segala jenis file.
Sedangkan Vim hanyalah pengembangan dari Vi, yang biasa dikenal dengan vim = vi improved yang juga sangat stabil dan dapat ditambahkan berbagai macam plugin.
Pada saat masuk ke program vi, maka kita akan berada pada mode command. Berkut beberapa perintah dasar dari editor vim/vi:

  1. i : untuk beralih ke mode insert atau memasukkan teks.
  2. Esc : tekan tombol esc untuk beralih ke mode command dari mode insert/input.
  3. dd : menghapus satu baris.
  4. /teks dan ?teks : fungsinya untuk mencari kata teks.
  5. u : undo perubahan terakhir.
  6. U : undo semua perubahan.
  7. D : menghapus hingga akhir baris.
  8. <ctrl>+d : scroll layar ke bawah.
  9. <ctrl>+u : scroll layar ke atas.
  10. :w : menyimpan file.
  11. :wq : menyimpan file dan keluar (quit).
  12. :q! : keluar dan tidak menyimpan file.
  13. bisa juga mengedit sebuah file dengan tambahan perintah: 
    # vi +15 file.conf
    • artinya mengedit file.conf pada baris ke-15
