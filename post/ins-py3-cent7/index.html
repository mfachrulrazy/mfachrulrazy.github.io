Install Python 3 di Centos 7

Install python 3.7.4 pada Centos 7

 Posted on May 7, 2019  |  1 minutes  |  MFR

Pada saat artikel ini dibuat, versi python terbaru adalah 3.7.4.
Install paket pendukung untuk python 3 

# yum install gcc openssl-devel bzip2-devel libffi-devel
Download python 3 dari situs resminya 
# wget https://www.python.org/ftp/python/3.7.4/Python-3.7.4.tgz
Pindah ke direktori dan install python 3 tersebut 
# cd Python-3.7.4
# ./configure --enable-optimizations
# make altinstall
Penggunaan make altinstall agar binary python default (/usr/bin/python) tidak diganti.
Hapus file download python.
Cek versi python 
# python3.7 -V

Python 3.7.4
Buat alias kustom agar lebih mudah memanggil python 3.7.4 ini. Edit file ~/.bashrc (jika menggunakan bash terminal) dan tambahkan baris berikut. 
...
alias python3='python3.7'
alias pip3='pip3.7'
...
Kemudian logout dan login kembali (memulai session baru)
