Kompress dan Ekstrak File di Centos

Kompress dan ekstrak file menggunakan beberapa tools

 Posted on January 11, 2019  |  2 minutes  |  MFR

Kompress File dengan Zip

Kompress file dengan menggunakan zip 

# zip -r namafile.zip nama_file
Penggunaan opsi -r berarti dilakukan rekursif di dalam folder dan subfolder

Ekstrak File dengan Unzip

Ekstrak file bisa menggunakan unzip.
Untuk melihat isi dari file zip tanpa meng-ekstrak terlebih dahulu 

# unzip -l namafile.zip
Ekstark file zip 
# unzip namafile.zip
Ekstrak dengan tujuan direktori tertentu 
# unzip namafile.zip -d /var/usr

Gzip dan Gunzip

Compress file dengan menggunakan gzip 

# gzip namafile.txt
Perintah di atas akan memberikan output berupa file namafile.gz.
Ekstrak file dengan menggunakan gunzip 
# gunzip namafile.gz

Kompress file dengan Tar

Kompress file dengan menggunakan tar dan gz 

#tar -zcvf /folder/file.tar.gz /folderA
Setelah perintah tar, masukkan lokasi dan nama file dengan ekstensi .tar.gz kemudian nama folder atau file yang ingin dikompress.

  • -z : kompress/ekstrak dengan gzip/gunzip
  • -j : kompress/ekstrak dengan bzip2/bunzip2
  • -c : buat archive di linux
  • -v : verbose atau menampilkan proses di linux
  • -f : nama file

Ekstrak file dengan Tar

Untuk ekstrak file hanya berbeda pada opsinya.

# tar -xvf /folder/file.tar.gz
Opsi -x untuk ekstrak file yang diberikan.
Jika ingin ekstrak di folder tertentu tambahkan opsi -C
# tar -xvf /folder/file.tar.gz -C /home/folderA

Opsi Tar Lainnya

Untuk melihat isi di dalam file tar bisa dengan menggunakan opsi t 

# tar tvf /folder/file.tar.gz

Tools kompress lain

Zcat

Menggunakan zcat untuk ekstrak file yang sama dengan ketika menggunakan perintah gunzip -dc opsi d untuk decompress dan c untuk mengirimkan hasil ke standard output. Contoh perintah: 

# zcat namafile.tar.gz | tar xvf -
Opsi (-) setelah parameter tar artinya sama dengan namafile yang diekstrak oleh zcat ke standard output.

bzip2 dan bunzip2

Sama dengan penggunaan gzip dan gunzip, hanya saja hasil kompressnya lebih kecil. Ekstensi yang digunakan adalah .bz2.
Dalam tar bunzip bisa digunakan dengan opsi j.

xz dan unxz

Sama seperti gzip, xz untuk compress file dan unxz untuk ekstrak file.
