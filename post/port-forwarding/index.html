Port Forwarding IP Publik pada Mikrotik

 Posted on July 5, 2017  |  1 minutes  |  MFR

Biasanya dalam sebuah jaringan kantor, memiliki IP Publik yang terbatas. Untuk memanfaatkan IP tersebut ke banyak server bisa menggunakan port forwarding.
1. Sebelum membuat port forwarding, pastikan port yang akan digunakan tidak ada dipakai oleh mikrotik di winbox pada menu IP > services
2. Setelah itu lakukan konfigurasi port forwarding di bagian IP > Firewall > NAT
3. Tekan tombol + > Tab General
4. Bisa di-isikan bagian In Interface adalah interface yang ke internet. 5. Tab Action

Jika lebih dari 1 IP Publik

Untuk IP yang lebih dari 1, harus dibuatkan kembali rule agar setiap akses pada server akan direspon oleh IP publik yang telah ditentukan
1. Buatkan rule baru srcnat
2. Tab General 3. Tab Action
