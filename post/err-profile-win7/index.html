Error User Profile Windows 7

Mengatasi Pesan Error User Profile Cannot Be Loaded pada Windows 7

 Posted on June 17, 2017  |  1 minutes  |  MFR

Salah satu komputer klien mendapat error tidak dapat me-load user profile, akibatnya semua pengaturan-pengaturan pribadi seperti shortcut di desktop tidak muncul. Untuk dapat memunculkan kembali user profile, lakukan langkah berikut
1. Restart komputer/laptop dan masuk ke dalam Safe Mode (menekan tombol F8 ketika booting)
2. Setelah login pada windows, buka Regedit
3. Masuk ke Folder HKEY_LOCAL_MACHINE >> SOFTWARE >> Microsoft >> Windows NT >> CurrentVersion >> ProfileList
4. Dalam folder tersebut terdapat 2 folder dengan awalan S-1-5-21 diikuti dengan nomor yang panjang dan sama persis di belakangnya. Tetapi yang membedakan adalah satu berakhiran .bak 5. Selanjutnya, ubah nama folder tanpa akhiran .bak dengan menambahkan akhiran .bk pada folder tersebut.
6. Kemudian, untuk folder yang berakhiran .bak ubah menjadi tanpa .bak
7. Pilih folder tanpa .bak yang baru saja sudah diubah namanya, kemudian pilih value:
RefCount dan ubah value data menjadi 0 (nol). State dan ubah value data menjadi 0(nol). 8. Jika sudah tutup program regedit dan restart kembali komputer/laptop. Sekarang user profile sudah bisa kembali.
