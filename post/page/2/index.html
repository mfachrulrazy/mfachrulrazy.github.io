Install Remote Desktop Ubuntu

 Posted on September 15, 2017  |  1 minutes  |  MFR

  1. Update paket Ubuntu, kemudian install paket xrdp 
    $ sudo apt-get install xrdp
  2. Install paket xfce4 
    $ sudo apt-get install xfce4
  3. Modifikasi xrdp berjalan startup file di ~/.xsession 
    $ echo xfce4-session >~/.xsession
  4. Edit startup file xrdp untuk mengeksekusi xfce4 service pada saat startup 
    $ sudo nano /etc/xrdp/startwm.sh
  5. Tambahkan: 
    1...
2startxfce4
3. /etc/X11/Xsession
  6. Sampai di sini, xrdp sudah bisa berjalan tetapi belum bisa login karena port 3350 belum “LISTENING” 
    $ sudo systemctl enable xrdp
  7. Kemudian reboot sistem
