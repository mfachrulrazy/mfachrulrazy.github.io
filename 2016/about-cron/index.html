Berkenalan dengan Cron

Written by MFR with ♥ on 5 February 2016 in server linux cron 1 min

Apa itu cron? Jadi, cron adalah sebuah software utilitas di lingkungan linux/unix yang berfungsi sebagai pembuat jadwal pekerjaan berdasarkan waktu yang bisa ditentukan.

Untuk membuat cron baru 

# crontab -e
Melihat daftar cron 
# crontab -l
Setiap perintah yang dijalankan di cron, akan mengirim ke email secara default dan biasanya dalam terminal akan muncul 
You have new mail in /var/spool/mail/root
Isi dari file /var/spool/mail/root adalah proses dari script yang dijalankan.

Untuk menghilangkan pesan ini muncul dapat dilakukan redirect output ke /dev/null. Karena outputnya berupa standard out maka nilainya adalah 2.

Edit file cron dan tambahkan pada akhir baris 

* * * * * sh /home/scriptcron.sh > /dev/null 2>&1
Dimana 5 tanda bintang itu merepresentasikan 
* * * * *
| | | | |---- hari dalam minggu (1-7)
| | | |------ bulan (1-12) 
| | |-------- hari dalam bulan (1-31)
| |---------- jam
|------------ menit
Inputan lain dari cron

  • * : nilai tiap detik (semuanya)
  • , : pemisah nilai
  • - : rentang waktu
  • / : langkah
  • 7 : SUN (non-standard)
  • SUN-SAT : untuk single value
