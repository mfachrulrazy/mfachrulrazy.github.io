Menampilkan Saved Password Wifi Windows 10

Written by MFR with ♥ on 18 September 2016 in windows networking 1 min

Menemukan password wifi yang sudah tersimpan (saved password) melalui command prompt dapat menggunakan tools netsh.

> netsh wlan show profiles "nama_wifi" key=clear

Hasilnya, dapat dilihat pada bagian Security Settings terdapat parameter Key Content dan isinya adalah password wifi tersebut.
Security settings
-----------------
    Authentication         : WPA2-Personal
    Cipher                 : CCMP
    Authentication         : WPA2-Personal
    Cipher                 : GCMP
    Security key           : Present
    Key Content            : PASSWORD_WIFI
